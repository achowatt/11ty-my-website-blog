I learned two different ways of setting up sass.
<!-- Excerpt End -->
Step 1:

npm init

Step 2:
npm install node-sass --save-dev

Step 3:

npm install

Step 4, in package.json, add:

"scripts": {
    "compile:sass": "node-sass sass/main.scss css/style.css"
}
Step 5:

npm run compile:sass

Step 6, adding -w (to watch the changes):
"scripts": {     "compile:sass": "node-sass sass/main.scss css/style.css -w" }
